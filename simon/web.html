En web te enseñan a como uno puede aprender a hacer una pagina web, con HTML, que es html? arrow_back_ios

HTML son las siglas de Hyper Text Markup Language. HTML es el lenguaje de marcado estándar, para crear
páginas web.
HTML describe la estructura de una página web. HTML consta de una serie de elementos. Los elementos HTML le dicen al navegador cómo mostrar el contenido.
Los elementos HTML etiquetan partes de contenido como "esto es un encabezado", "esto es un párrafo", "esto es un enlace", etc.
